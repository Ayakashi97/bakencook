# Administration Guide

## Access
Access to the Admin Dashboard is restricted to users with the `ADMIN` role. The initial admin user is created via the **Onboarding Wizard** when the application is first launched.

## User Management
- **List Users**: View all registered users.
- **Manage Roles**: Promote users to Admin or demote them to standard Users.
- **Deactivate**: Deactivate user accounts to prevent login.

## System Settings
- **Global Configuration**: Manage system-wide settings such as default units, language preferences, and feature toggles.
- **Ingredients Database**: Manage the master list of ingredients and their properties (e.g., hydration, calories).
- **Units**: Configure available units of measurement and their conversions.

## Server Management
- **Status**: Monitor the health of the backend services and database connection.
- **Logs**: View system logs for debugging purposes.
